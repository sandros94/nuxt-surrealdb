# Other Composables

Beyond the main composables, Nuxt SurrealDB provides additional SSR-safe wrappers for common SurrealDB operations.

## useSurrealRun

Execute a SurrealDB function by name.

```vue
<script setup lang="ts">
const { data } = await useSurrealRun('fn::my_function', ['arg1', 'arg2'])
</script>
```

| Argument | Type | Description |
|----------|------|-------------|
| `name` | `MaybeRef<string>` | Function name |
| `args?` | `MaybeRef<unknown[]>` | Function arguments |
| `asyncDataOptions?` | `AsyncDataOptions & { key?: string }` | Options for `useAsyncData` |

## useSurrealVersion

Retrieve the SurrealDB server version.

```vue
<script setup lang="ts">
const { data: version } = await useSurrealVersion()
</script>
```

| Argument | Type | Description |
|----------|------|-------------|
| `asyncDataOptions?` | `AsyncDataOptions & { key?: string }` | Options for `useAsyncData`. Default key: `'surreal:version'` |

## useSurrealExport

Export the database as a SurrealQL string.

```vue
<script setup lang="ts">
const { data: dump } = await useSurrealExport({})
</script>
```

| Argument | Type | Description |
|----------|------|-------------|
| `expOptions` | `MaybeRef<SqlExportOptions>` | Export options |
| `asyncDataOptions?` | `AsyncDataOptions & { key?: string }` | Options for `useAsyncData` |

## useSurrealImport

Import a SurrealQL dump.

```vue
<script setup lang="ts">
const { data: success } = await useSurrealImport(surrealqlDump)
</script>
```

| Argument | Type | Description |
|----------|------|-------------|
| `input` | `MaybeRef<string>` | The SurrealQL string to import |
| `asyncDataOptions?` | `AsyncDataOptions & { key?: string }` | Options for `useAsyncData` |

## useSurrealMemory

Access the in-memory WASM SurrealDB client. Requires `@surrealdb/wasm`.

```vue
<script setup lang="ts">
const client = await useSurrealMemory()

if (client) {
  const [results] = await client.query('SELECT * FROM test;').json().collect(0)
}
</script>
```

Returns `Promise<Surreal | null>` — `null` if `@surrealdb/wasm` is not installed.

::tip
An alias `useSurrealMem` is also available.
::

## useSurrealLocal

Access the local embedded WASM SurrealDB client. Requires `@surrealdb/wasm`.

```vue
<script setup lang="ts">
const client = await useSurrealLocal()

if (client) {
  const [results] = await client.query('SELECT * FROM test;').json().collect(0)
}
</script>
```

Returns `Promise<Surreal | null>` — `null` if `@surrealdb/wasm` is not installed.

::warning
The IndexedDB WASM engine is currently bugged upstream. In-memory mode works as expected.
::
