# useSurrealAuth

Fetch the currently authenticated user's information via `client.auth()` with SSR support.

## Usage

```vue [app/pages/profile.vue]
<script setup lang="ts">
const { data: user } = await useSurrealAuth()
</script>

<template>
  <div v-if="user">
    Logged in as: {{ user.name }}
  </div>
</template>
```

## Arguments

| Argument | Type | Description |
|----------|------|-------------|
| `asyncDataOptions?` | `AsyncDataOptions & { key?: string }` | Options passed to `useAsyncData`. Default key: `'surreal:info'` |

## Return type

Returns the same shape as [`useSurrealAsyncData`](/composables/use-surreal-async-data), with `data` typed as the authenticated user record or `undefined`.
